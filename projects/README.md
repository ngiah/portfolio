# Project Content

Each project has its own folder for screenshots, videos, reports, and source-code links.

## Add or update a project

1. Put project media inside its matching folder.
2. Use a landscape cover image named `cover.jpg` or `cover.png`.
3. Update the matching object in `projects.json`.
4. Set `cover` to the image path, for example:

   `"cover": "projects/plc-monitoring-system/cover.jpg"`

5. Add public URLs under `links` when available.

GitHub Pages cannot automatically scan folders, so `projects.json` acts as the project index.
