# Vo Xuan Nghia Portfolio

Static portfolio website for Vo Xuan Nghia, Automation Engineering Intern.

## Run locally

Open `index.html` directly in a browser.

## Deploy

This site is ready for GitHub Pages, Netlify, Vercel, or any static hosting service.

## Add project content

Project cards are generated from `projects/projects.json`.

Put project images and documents inside the matching folder under `projects/`.
Use `projects/_project-template/` when adding a new project.

The selected Canva showcase design can be edited at:
https://www.canva.com/d/xmxPmZjPpeQ3DKu
